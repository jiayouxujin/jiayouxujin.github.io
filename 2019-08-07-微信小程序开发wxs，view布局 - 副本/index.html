微信小程序view:flex布局

使用flex布局显示声明 display:flex;

做一个简单的demo

现在我们都加上display:flex 因为view不会自动继承，需要在每个想使用的view里加上

首先是横向布局和竖向布局,属性 flex-direction

  • row 从左到右的水平方向为主轴
  • row-reverse：从右到左的水平方向为主轴
  • column：从上到下的垂直方向为主轴
  • column-reverse：从下到上的垂直方向为主轴

注意：因为不会继承，发现row只要加到item1 item2 item3即可(错误)，而row-reverse 要全部都加

接着是横向上的布局方向，属性justify-content

  • flex-start：主轴起点对齐(默认值)
  • flex-end：主轴结束点对齐
  • center：在主轴中居中对齐
  • space-between：两端对齐，除了两端的子元素分别靠向两端的容器之外，其他子元素之间的间隔都相等
  • space-around：每个子元素之间的距离相等，两端的子元素距离容器的距离也和其它子元素之间的距离相同

最后是纵向上的布局方向，属性align-items属性

  • stretch 填充整个容器(默认值)
  • flex-start 侧轴的起点对齐 （这里我们手动设置下子 view 的高度，来看的明显一些）
  • flex-end 侧轴的终点对齐
  • baseline 以子元素的第一行文字对齐

补充

  1. align-self属性覆盖父元素的align-items属性
  2. flex-warp属性控制warp是否换行
  3. order属性，控制排列顺序
