操作DOM

  • text() 设置或者返回选中元素的text内容
  • html()设置或者返回选中元素的html内容 包括html 标签
  • val() 设置或者返回form里的值
  • attr()得到属性里的值

刚刚上面我们只是得到值，现在我们进行操作–设置值。同样的我们还是使用text() html() val()三种函数

以上四个函数都有callback function 其中有两个参数 一个是表示当前选中的元素的index，还有一个表示原来的值

添加元素

  • append() 在当前选中的元素后面添加新的元素
  • prepend() 在当前选中的元素前面添加新的元素

删掉元素

  • remove() 去掉选中的元素包括其子元素
  • empty() 去掉选中元素子元素

###Get and Set Css

  • addClass() 给选中元素添加一个class
  • removeClass()给选中元素移除一个calss
  • toggleClass()
  • css() 设置style属性
