前言

每一天都要比前一天技术多学一些，更新自己的技术库

AJAX介绍

通过AJAX你可以完成

  • 在页面加载完成后，从服务器读取数据
  • 不需要重新加载页面，就能更新局部页面
  • 在后台给服务器发送数据

XMLHttpRequest对象

所有现代浏览器都支持XMLHttpRequest对象，能够用来跟服务器交换数据，也就是说需要要重新加载页面，就能够更新局部页面

发送请求

使用open()和send()方法

解释一下上面的代码

  1. open(method,url,async) method:表示请求的类型GET或者POST url请求的地址 async true表示异步 false表示同步
  2. send()发送请求给服务器，多用于GET请求
  3. send(String)发送给服务器，多用于POST请求

这里简单再讲一下异步或者同步

  • 同步请求，表示当该请求得到response的时候，才进行其他请求
  • 异步请求，表示不一定等待该请求得到结果

两者的不同点已经很清楚了，不过呢，同步请求只在一些特殊场合使用，例如你需要使用得到请求结果的值在发起下一个请求，不然对于服务器比较慢或者卡顿严重的，会影响体验。

处理请求 onreadystatechange

当请求得到应答的时候，执行该方法

  • onreadystatechange 当readystate参数发生改变的时候执行
  • readyState 0request not initialized 1 server connection established 2 request receved 3 processing request 4 request finished and response is ready
  • status 200 OK …
  • statusText “OK”

封装

返回结果

  • responseText 让response data 变成string
  • responseXML 让response data 变成xml data

以上我们学习的javascript的ajax。接下来我们学习jQuery AJAX

JQury介绍

让你在网页使用javascript更简单，是一个”write less,do more”的Javascript库。

通过使用jQuery AJAX方法，你可以从服务器请求 text,HTML,XML或者JSON通过HTTP Get和HTTP Post

GET vs POST

$.get(url,callback)方法通过使用HTTP Get请求从服务器请求到数据

$.post(url,data,callback)方法通过使用HTTP Post请求向服务器发送请求

###Ajax()

至此，都可以可以通过ajax来请求服务器的数据，这样的话就可以把一个静态页面变活。

其他收获

定时刷新 settimeout()

语法 setTimeout(function,milliseconds,param1….)

解释一下

  • function required并且是可以被执行的
  • milliseconds 1000=1second
