不同人修改了不同文件如何处理

第一个人

  1. 现在github的master分支上新建一个feature/function
  2. git clone ssh gitname
  3. git config –local user.name ‘newusername’
  4. git config –local user.email ‘newemail’
  5. git branch -av
  6. git checkout -b branchname basebranchname
  7. git push origin localbranch:remotebranch

第二个人

  1. git fetch
  2. git branch -av
  3. git checkout -b branchname basebranchname
  4. git add .
  5. git commit -m
  6. git fetch
  7. git merge branchname
  8. git push

不同人修改同一文件的不同区域

第一个人

  1. git pull 更新
  2. 修改
  3. git push

第二个人

  1. git pull
  2. git merge branchname
  3. git push
