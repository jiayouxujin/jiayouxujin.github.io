Linux命令

发表于 2020-03-09 | 更新于: 2020-04-21 |
字数统计: 1.2k | 阅读时长 ≈ 4

系统操作

帮助命令

  • man命令，man命令一共有9章，代表所查询的命令不同的格式，例如：man ls这个代表的是shell命令行，man 5 passwd代表的是passwd这个文件。

man passwd默认是第一章，shell脚本命令.

  • help命令也是常用的帮助命令，不过该命令有两种形式

help cd

ls –help

这两种格式的原因是因为内外部命令之分，自己直观的感觉是cd help会跑到help的文件夹下，如果执行cd --help则会报错

区分内外部的命令type 命令

  • info命令是比help更加详细的内容,其格式是:info ls

文件操作

  • pwd显示当前所在的文件目录下，并且pwd是内部命令。使用help的时候要注意。【这里还要区分两个路径 /和/root,前者是根目录，后者是root的家路径】

  • ls显示当前目录下的所有文件，并且有丰富的option.这里举几个简单的option.

-l表示查看详情,-a表示显示所有文件，包括隐藏文件，-r进行排序,-t通过时间进行排序,-R递归显示所有文件夹，包括里面的文件

  • cd操作符，可以用来跳转当前目录，常用的格式,参见截图

  • mkdir创建新的文件，可以通过-P，创建多级目录
  • rm -r递归删除文件，但是会提示时候进入文件夹，-f这个Option则不会提示，会强制删除，这个删除文件的操作一定要小心！！

  • cp就是copy的简写，对文件进行复制。
  • mv对文件进行重命名，但是在linux中,重命名就相当于移动文件！
  • cat进行文本查看，head只显示头部,tail显示尾部，wc统计文本信息,例如wc -l filename统计文本的行数
  • tar首先需要懂得Linux中对文件有两步操作，1.打包成文件2.进行压缩，tar cf进行的是打包成文件，tar czf把文件进行打包并且使用gzip算法进行压缩，tar cjf把文件进行打包并且使用bzip2进行压缩。
  • tar xf对打包的文件进行解包，并且通过-C 可以将其移到想要的地方。如果是gzip算法压缩的后缀是.gz那么可以通过tar xzf，如果是bzip2算法进行压缩的，可以通过tar xjf

用户和权限管理

su - user1临时切换用户，-会把文件夹定位到对应用户的家目录

visudo可以给某个用户赋予权限，如果不知道某个命令在哪里，通过执行:!which 命令

文件权限

  • 文件类型：-普通文件,d目录文件，b块特殊文件,c字符特殊文件
  • 字符权限r4,w2,x1
  • ls -ld查看文件夹的权限，ls -ld查看文件的权限
  • chmod修改权限,chmod 777 filename,，chown修改文件所属

网络管理

网络状态查看

  • net-tools

    • ifconfig

      • eth0第一个网卡的名称

      inet是网卡的Ip地址，ehter网卡的mac地址

      • 网络接口命名修改

    • route

      • 查看网关route -n

    • netstat

网络故障排除命令

  • ping检测当前主机和目标主机是否畅通
  • traceroute辅助ping,如果目标主机畅通，但是还是不能用，可以检测中间路由，traceroute -w 1 www.baidu.com
  • mtr数据包是否丢失
  • nslookup域名对应的ip
  • telnet检查端口的连接状态 telnet www.baidu.com 80
  • tcpdump细致数据包
  • netstat -ntpl可以查看当前主机上，对外开放的端口

软件包管理

  • CentOS、RedHat使用的yum包管理器，软件安装包格式为rpm
  • Debian、Ubuntu使用的apt包管理器，软件安装包格式为deb

操作

更改yum源

  • yum install
  • yum list
  • yum remove
  • yum update

源代码安装

  • wget
  • tar -zxf
  • cd direction
  • ./configure 自动配置内核版本 –prefix安装的位置
  • make -j2 表示我要用两个逻辑cpu进行编译
  • make install

进程概念

管理程序开始到终止的整个生命周期

查看命令

  • ps(进程状态)

当前shell运行的进程

ps -eLf|more基础用法就是这几个

  • pstree(进程树形结构)
  • top

20days表示最近开机运行的天数

load average：分别表示1分钟5分钟10分钟

进程间的通信

信号是进程间通信的方式之一，常用的方法是终端用户输入中断命令，通过信号机制停止一个程序的运行

  • kill -l

  • kill -9 PID 不受影响关掉进程

守护进程

  • 使用nohup 与& 后台挂起，忽略hanup挂起信号【这个不是守护进程】

ps -el |grep tail

服务管理工具systemctl

  • systemctl start|stop|restart|status 服务名字

shell脚本

shell是命令解释器，用于解释用户对操作系统的操作。提供一个与用户对话的环境。
